🤖 AI Bias Detection in Hiring

Promoting fair and inclusive hiring through AI-powered bias detection

🚀 Live Demo

This Flask web application detects bias in hiring processes and provides fair candidate evaluation.

🌐 View Live Demo 📂 View Source Code

✨ Key Features

🔍

Bias Detection

Identifies gender, age, personality, and ethnicity bias in job descriptions

🔒

Resume Anonymization

Removes personal identifiers to focus on qualifications and skills

🎯

Skill Matching

Calculates candidate-job compatibility based on technical skills

📊

Performance Analytics

Confusion matrix showing system accuracy and performance metrics

📈 Project Impact

  • ✅ Promotes fair hiring practices by removing unconscious bias
  • ✅ Focuses on skills and qualifications rather than personal attributes
  • ✅ Helps companies build diverse teams and meet inclusion goals
  • ✅ Provides data-driven insights for HR decision making

🚀 How to Run Locally

# Clone the repository
git clone https://github.com/linga-1221/ai-for-reducing-bias-in-hiring.git

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py

# Open browser at http://127.0.0.1:5003
